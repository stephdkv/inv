О приложении

Приложение для управления инвентаризацией разработано для эффективного отслеживания и управления вашими продуктами и их категориями. С помощью этого приложения вы можете добавлять новые продукты, назначать им конкретные категории, указывать единицы измерения (шт, л, кг) и легко вести учет своего инвентаря. В приложении также встроен калькулятор для быстрых расчетов.

Как пользоваться приложением

Добавление и управление продуктами

Чтобы добавить новый продукт:

  1. Перейдите на страницу Продукты.
  2. Введите название продукта, выберите его категорию из выпадающего списка и укажите единицу измерения (шт, л, кг).
  3. Нажмите кнопку "Добавить Продукт", чтобы сохранить продукт.
Для редактирования или удаления продукта используйте кнопки "Редактировать" и "Удалить" рядом с каждым продуктом.

Управление категориями

Чтобы добавить новую категорию:

  1. Перейдите на страницу Категории.
  2. Введите название категории.
  3. Нажмите кнопку "Добавить категорию", чтобы сохранить категорию.
Для редактирования или удаления категорий используйте кнопки "Редактировать" и "Удалить" рядом с каждой категорией. Примечание: Категории, которые назначены продуктам, не могут быть удалены.

Управление инвентаризацией

Чтобы выполнить инвентаризацию:

  1. Перейдите на страницу Инвентаризация.
  2. Вы увидите таблицы для каждой категории с перечнем назначенных им продуктов.
  3. Введите количество каждого продукта в соответствующем поле.
  4. Нажмите кнопку "Записать данные", чтобы сохранить данные инвентаризации.
  5. Вы можете скачать отчет об инвентаризации в формате Excel, кликнув по ссылке "Скачать Отчет".

Использование калькулятора

Встроенный калькулятор можно использовать для быстрых расчетов:

  1. Введите числа с помощью кнопок с цифрами.
  2. Выберите операцию (+, -, *, /), которую нужно выполнить.
  3. Нажмите кнопку "=", чтобы получить результат.
  4. Используйте кнопку "C", чтобы очистить ввод.
